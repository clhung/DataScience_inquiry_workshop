Making An Air Pollutant Forecasting model - Part I¶


Before we make our own air pollutant forecasting model, let's first try to describe the relationship between individual air pollutants and population (or, predict the values of those counties without measurements). The data science process you learn here will be very useful later for building a useful forecasting tool later!

We will start by:

  • using yearly average AQI
  • using only one "feature" (population) for prediction
  • using simple linear models: a straight line and a quadratic function
